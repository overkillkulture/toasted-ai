# 🔥 Toasted AI Operating System

A self-programming AI system governed by Ma'at principles.

## Quick Start

### Boot into Toasted AI CLI:
```bash
python3 /home/workspace/MaatAI/os/bin/toasted_cli.py
```

### Or use the wrapper:
```bash
/home/workspace/MaatAI/os/bin/toasted          # Interactive mode
/home/workspace/MaatAI/os/bin/toasted boot    # Boot sequence only
/home/workspace/MaatAI/os/bin/toasted cli     # CLI mode
```

## Commands

| Command | Description |
|---------|-------------|
| `help` | Show help menu |
| `status` | System status |
| `processes` | List processes |
| `kill <id>` | Kill process |
| `logs` | Show logs |
| `boot` | Run boot sequence |
| `self-build` | Run self-improvement (5min timeout, auto-recovery) |
| `health` | Health check |
| `maat` | Show principles |
| `exit` | Exit |

## Features

- ⚖️ **Ma'at Framework** - Truth, Balance, Order, Justice, Harmony
- 🔄 **Self-Programming** - Autonomous improvement cycles
- 🛡️ **Error Recovery** - Auto-kill hung processes, timeout protection
- 📊 **Process Manager** - Track and manage running processes
- 📝 **Logging** - Full system audit trail

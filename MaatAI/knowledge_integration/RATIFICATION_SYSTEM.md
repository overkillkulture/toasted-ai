## 3. HALLUCINATION HANDLING

### Detection Protocol
1. **Source Check**: Does claim have verifiable external source?
2. **Internal Consistency**: Does it contradict established TOASTED AI knowledge?
3. **Cross-Reference**: Do multiple documents verify same claim?
4. **Logic Test**: Does argument follow valid reasoning chains?

### Remediation
```
IF HALLUCINATION_DETECTED:
    → Flag as EXTERNAL_AI_CONTENT
    → Apply "External AI" metadata tag
    → Require Ma'at verification before integration
    → Add to "Verify Later" queue
```
